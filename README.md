# wasmavelli
Rust [Machiavelli](https://en.wikipedia.org/wiki/Machiavelli_(Italian_card_game)) (with Jokers)
solver targeting WASM. You can use it from my [personal website](https://dede1751.github.io/#machiavelli).

This project is the result of a friendly dispute at the end of a game, where we could not determine
if a win was possible. As there were no existing solvers that also accounted for jokers, I decided
to code one myself.
